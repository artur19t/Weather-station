# Weather Station

Этот проект представляет собой метеостанцию на базе **STM32F401** с использованием интерфейсов **SPI** и **I2C** для подключения дисплея и датчиков.

На текущем этапе:
- Интерфейсы **SPI** и **I2C** полностью сконфигурированы.
- Настройки выполнены в **CubeMX** и дополнены кодом на **LL-библиотеках**.

---

## I2C 
**I2C1** настроен и готов к работе:
- Скорость: 100 kHz 
- Pull-up подтяжки включены (SCL, SDA)
- Режим: I2C
- Генерация START/STOP и обработка флагов — через LL

### Основные моменты настройки:
- **Pinout:**
  - PB6 → I2C1_SCL
  - PB7 → I2C1_SDA
- **Clock:**
  - I2C1 тактируется от APB1
- **Настройки I2C:**
  - Addressing Mode: 7-bit
  - Dual Address: Disabled
  - No-Stretch: Disabled
  - General Call: Disabled

---

## SPI 
**SPI1** сконфигурирован для дисплея:
- Режим Master
- CPOL и CPHA выставлены согласно даташиту дисплея
- Скорость шины подобрана под ограничение дисплея
- Управляющие пины (CS, DC, RESET) вынесены на GPIO
